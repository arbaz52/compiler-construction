﻿# SLR(1) Parser
- Creates a NFA for the given grammar
- Creates a parsing table for the given grammar

## Notes
- Can handle epsilons '~'

## How to use
- Open the application
- Provide a grammar (some example grammars are provided in production_rules.txt file)
